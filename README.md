# AI Interview Platform 🚀

An AI-powered mock interview platform built using modern web technologies and OpenAI API.

---

# 🌟 Features

✅ AI Mock Interviews  
✅ Resume Based Questions  
✅ Frontend / Backend / DSA Interviews  
✅ AI Voice Questions  
✅ Speech Recognition  
✅ Authentication System  
✅ Protected Routes  
✅ Dashboard Analytics  
✅ Interview History  
✅ MongoDB Database  
✅ Real-time AI Feedback  
✅ Score System  

---

# 🛠 Tech Stack

## Frontend
- React.js
- Tailwind CSS
- React Router DOM

## Backend
- Node.js
- Express.js

## Database
- MongoDB Atlas

## AI
- OpenAI API

# ⚙️ Installation

## Clone Repository

git clone https://github.com/Ahad4562/ai-interview-platform.git

---

# 🚀 Frontend Setup

cd client
npm install
npm run dev

---

# 🚀 Backend Setup

cd server
npm install
npm start

---

# 🔑 Environment Variables

Create a `.env` file inside `server/`

OPENAI_API_KEY=your_openai_api_key
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
